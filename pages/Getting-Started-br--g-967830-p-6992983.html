iThenticate at Caltech

iThenticate ( https://www.ithenticate.com/ ) is a text similarity checking tool that compares documents (including journal article manuscripts, proposals, research reports, thesis, and dissertations, etc.) against millions of published works, subscription databases, and web pages.

Caltech licenses iThenticate for use by Faculty, Post-Docs, and Graduate Students to use as a self-check on manuscript quality before submission.

Eligible users may access iThenticate from Caltech Access site - look for the link under Academic Services. ⇒ Important note: Caltech's IMSS manages campus access to Ithenticate.

Usage Policy: Account holders are granted access in order to upload and perform text analysis on one's OWN DOCUMENTS ONLY.

File and System Requirements in iThenticate

Accepted File Types

Please note that LaTeX is NOT currently accepted.

  • Microsoft Word® (DOC and DOCX)
  • Word XML
  • Plain Text (TXT)
  • Adobe PostScript®
  • Portable Document Format (PDF)
  • HTML
  • Corel WordPerfect® (WPD)
  • Rich Text Format (RTF)

Accepted File Sizes

  • Files must be less than 100Mb
    • Files of larger size may be reduced in size by removal of non-text content.
  • The maximum document length is 800 pages.
  • Files must contain at least 20 words of text.
  • Files must not exceed 2 Mb of raw text.
  • Zip files may contain up to 100 Mb or 1000 files.
    • A zip file to be uploaded may not exceed either limit.
    • Zip files should be checked to ensure only usable file formats are included in the upload.
  • Files that are password protected, encrypted, hidden, system files, or read only files cannot be uploaded or submitted to iThenticate
