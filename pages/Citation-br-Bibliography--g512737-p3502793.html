About Bibliographies in Overleaf - Using BibTeX

BibTeX is a bibliography tool and style format that works in conjunction with TeX/LaTex. It has a similar "tag"-like format that assigns labels to different parts of a bibliographic citation - i.e. author, title, etc. - which should be familiar if you've ever used a program like Zotero, EndNote, or RefWorks. You can write your own BibTeX file from scratch, but most citation databases and publisher platforms allow you to download a citation directly from their pages. If you don't copy and paste individual citations directly into Overleaf (as shown in the video below), it is recommended to collect all of your citations in one file (i.e. by compiling in a citation manager program and then exporting) and then uploading that file to Overleaf.

From the Overleaf Tutorial, How to include a bibliography using bibtex :

You can easily manage your references in LaTeX using a bibliography, stored as a .bib file. This video shows how to create a new .bib file, add references from e.g. Google Scholar, and cite these references in your LaTeX document.

Converting EndNote References into BibTeX

You can export either a group (which could be your entire library) or selected references within a group to a BibTeX-formatted text file. You can also combine several export files into one BibTeX file to import into Overleaf. One recommended step before exporting references is to enter something in the Label field for each reference you plan on using in Overleaf. The Label field is found in all reference types in EndNote, and this is what you will use to find and select references in Overleaf. The Label field is found nearer the end of the reference field entry:

After labeling your references, select "BibTeX Export" under the Preview window on the right, then select a folder or a group of references within the folder to export to a BibTeX text file. Select "File" then "Export.." If you wish to export an entire folder, uncheck "Export Selected References" , and that will export all the references in the selected folder. Leave it checked to export only selected references.

Once you have your BibTeX file, change the file extension from .txt to .bib. You are now ready to upload your BibTeX bibliography file into Overleaf.

Adding a .bib File Into Overleaf

From within Overleaf, click "Add Files..." at the top left, then select Bibliography, then select "Upload a .bib file". Then select the .bib file you created from your EndNote export.
