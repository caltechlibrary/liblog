Exporting References from SciFinder

You can export a citation for an individual reference from the REFERENCE DETAIL screen, or you can select a number of references from the REFERENCES search results screen.

  1. In either case, select Export at the top right.
  2. Under For: Citation Manager select Citation export format (*.ris).
  3. Under Details: File Name * enter something you'll recognize.
  4. Click Export.
  5. Depending on the citation manager that you have installed, you will be given the option to open the file, or save it. EndNote and Zotero can open *.ris files and automatically import them.
