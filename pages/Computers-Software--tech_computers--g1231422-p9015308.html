Computers & Software

Computers

The Library provides Windows and Mac desktop and notebook computers for the campus community. These are all equipped with a suite of the most commonly needed software .

Windows workstations are located in Sherman Fairchild, Cahill, Dabney, Geology and Caltech Hall Lookout (9th floor).

Windows and Mac notebook computers are available for checkout in Sherman Fairchild.

Software

This page lists software available on Library computers: Windows desktops, Windows notebooks, and Macbook Pro notebooks. Most software titles are available on all these platforms and in all Library locations. Desktops with the (+) designation include additional design and 3D printing software. The Windows NOTEBOOK01 includes the ACerS-NIST Phase Equilibria Diagrams database and the ICDD PDF-4 database of inorganic diffraction data.
