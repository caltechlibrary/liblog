CAS Source Index (CASSI) Search Tool

  • CAS Source Index (CASSI) Search Tool
    Contains a listing of publications indexed by CAS since 1907, including serial and non-serial scientific and technical publications. Search by Title, Abbreviation, CODEN, ISBN, or ISSN. Limit of 100 answers per search. It compliments the print CASSI [Z5523 A524] which includes library holdings, etc. Provided by CAS as a public service. More about CASSI.
  • Beyond CASSI
    Journal title abbreviations from early chemical literature and other historical reference sources that may not be listed in the CASSI Search Tool. Includes Beilstein (1910) / Gmelin (1926) journal abbreviations - complete lists were published in Beilstein (Main Work, v.1, p. XXV) and Gmelin (8th ed. System #1, p. XXV-XXXI).. Compiled by Marion Peters, UCLA Librarian Emeritus.
