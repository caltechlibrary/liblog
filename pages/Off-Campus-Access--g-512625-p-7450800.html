How to access electronic resources from off-campus

Can I access Caltech Library resources (journals, ebooks, databases) from off campus?

Off-campus access to electronic resources (ebooks, electornic journals, databases, etc.) is restricted to members of the Caltech community with valid access.caltech (IMSS) credentials . If you are accessing material through a Library website (such as the Library Catalog or a LibGuide), you should see an orange "CaltechConnect" button. Click on this button to be redirected to the Library Proxy to enter your credentials.

Please see Off-Campus Access to Library Resources for more information.
