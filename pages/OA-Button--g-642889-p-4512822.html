What Is UnPaywall?

The Open Access Button is a Web discovery tool tool that looks for free, legally-uploaded research articles and datasets using a digital identifier (DOI, URL, PubMedID). It also serves as a reprint request button to communicate with authors that you would appreciate a copy of their research. If the work is made available via OA Button, it will be deposited in an open access repository available worldwide.

Testing by the Caltech Library indicates that the tool does not always find open access copies of subscription articles when we know they exist. But it does work some of the time, and that is encouraging ! We recommend adding Unpaywall to the Open Science toolkit for continued comparison with other discovery tools  -- this is a rapidly evolving landscape!
