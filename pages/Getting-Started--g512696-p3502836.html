Editing Account Preferences

After logging in, click the Preferences link on the top right of the search page.

On this page, you can set a few default options for your account:

  • Remove Duplicate References: By default, SciFinder returns results from both the CAPLUS and MEDLINE databases. Some articles are indexed in both of those databases - checking the box will return only the CAPLUS records for articles indexed in both.
  • Starting Page: By Default, the starting page is set to "Explore References". You can change that here.

Explore References Tips

  • CAS Standard Abbreviations & Acronyms
    Contains the abbreviations and acronyms that are automatically generated for terms in the abstract text, the keyword phrases, and the text modifying phrases for index entries in the SciFinder/CAplus database.
